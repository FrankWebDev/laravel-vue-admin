H5 自适应苏州实时公交查询系统
===============

> PHP 的运行环境要求 PHP7.0 以上。
1. PHP >= 7.0
2. PDO PHP Extension
3. MBstring PHP Extension
4. CURL PHP Extension

## 安装方法：
```php
git clone https://gitee.com/lisgroup/vueBus.git
composer install
```

## 使用方法
浏览器访问： http://域名 ，可以查看

在输入框输入查询的公交车，（如：快1）点击搜索后，会出现搜索到的车次，再次点击需要查询车次的方向，即可查看实时公交状态。
