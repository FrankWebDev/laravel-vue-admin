# RESTFull API 记录开发过程

## 开发步骤
```bash
php artisan make:controller Api/CronTaskController --resource

```

添加构造验证中间件
```php
/**
 * Create a new AuthController instance.
 * 要求附带email和password（数据来源users表）
 *
 * @return void
 */
public function __construct()
{
    // 验证登录
    $this->middleware('auth:api', ['except' => ['login']]);
}

```
