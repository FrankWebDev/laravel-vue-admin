# vue-admin

## 安装

```bash
# Clone 项目
git clone https://gitee.com/lisgroup/vueBus.git

# 切换目录
cd vueBus/admin

# 安装 npm 包
npm install

# 启动测试服务，默认 http://localhost:9090
npm run dev

# Build for production with minification
npm run build

# Build for production and view the bundle analyzer report
npm run build --report
```
